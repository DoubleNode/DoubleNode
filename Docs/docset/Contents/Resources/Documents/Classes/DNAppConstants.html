DNAppConstants Class Reference

Inherits from NSObject
Declared in DNAppConstants.h

Overview

Provides access to App-specific constants, specified in a plist file specified by the ‘Constants_plist’ setting in the App’s main *-Info.plst file.

fontConstant:

Creates and returns a UIFont object, initialized with the NSString objects loaded from the Constants plist file for the specified key.

+ (UIFont *)fontConstant:(NSString *)key

Parameters

key

The string which specifies the key to the Constants plist file.

Return Value

A new UIFont object, configured according to the value in the Constants plist file.

Discussion

key + Name is the generated key for loading the font postscript name.

key + Size is the generated key for loading the font size in “retina” points. This value is automatically divided by 2 before being used. This allows you to pull the font size points directly from a retina-sized Photoshop file.
